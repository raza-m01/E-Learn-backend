# 📚 E-Learn Backend

A Spring Boot application for managing an E-Learning platform.  
Supports courses, categories, videos, and user management, with JPA/Hibernate for ORM.

---

## 🚀 Tech Stack
- ⚙ **Java 21** 
- 🌱 **Spring Boot** (3.5.3)
- 🛠 **Spring Data JPA / Hibernate**
- 🐬 **MySQL** 
- 🛡 **Lombok** for reducing boilerplate
- 🔄 **Maven** for build & dependencies
- 🔥 **JUnit** / **Mockito** for testing
